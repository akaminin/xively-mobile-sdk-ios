Detailed Description

This class initiates authentication towards Xively system.

This class executes the username/password based authentications towards Xively. It returns a XISession if the authentication was successfull.

First you need to create an instance and set its delegate. Than you need to call XIAuthentication::requestLoginWithUsername:password:accountId:. If the parameters are correct, its result is called back on XIAuthenticationDelegate::authentication:didCreateSession: or XIAuthenticationDelegate::authentication:didFailWithError: whether it was a success or failure. The successfull branch return the valid XISession object that can be used to proceed further.
