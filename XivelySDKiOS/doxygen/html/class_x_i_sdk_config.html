Detailed Description

Configurations to set up the internal behavior of the SDK.

The session config is for fine tunning the SDK behavior. At this moment network handling parameters can be set here.

Since
Version 1.0

§ httpResponseTimeout

- (long) httpResponseTimeout
readnonatomicassign

The timeout for any HTTP request in seconds.

It is the overall timeout for any HTTP call which applies disregarding the retry attempt count and the waiting time between them. The default value is 15s.

Since
Version 1.0
